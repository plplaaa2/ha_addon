# 📓 Changelog

Korea Radio for Home Assistant의 모든 변경 사항을 기록합니다.

---

## 🚀 [2.0.1] - 2026-04-23
### 🌟 마이너 업데이트 (Minor Release)
이번 버전은 SBS 파싱 문제를 해결한 버전입니다.

## 🚀 [2.0.0] - 2026-04-22
### 🌟 메이저 업데이트 (Major Release)
이번 버전은 홈어시스턴트 생태계와의 완벽한 통합과 사용자 경험 고도화에 집중한 대규모 업데이트입니다.

#### 🔊 멀티 스피커 통합 & 스트리밍
- **미디어 플레이어 연동**: 홈어시스턴트에 등록된 모든 스피커 기기(Google Home, Sonos 등)로 라디오 송출 기능 추가.
- **내부 IP 자동 감지**: Supervisor API를 통해 호스트의 실제 내부 IP를 감지하여 스피커 접속 안정성 확보.
- **기기 자동 정지 및 전환**: 기기 선택 변경 시 이전 기기를 자동으로 끄고 새 기기로 즉시 재생을 전환하는 지능형 로직 구현.

#### 📱 동적 미디어 컨트롤러 (UI/UX)
- **듀얼 모드 UI**: 선택된 기기에 따라 브라우저 플레이어 또는 스피커 리모컨 UI로 자동 전환.
- **커스텀 리모컨**: 스피커 재생 시 사용할 수 있는 전용 Play/Pause/Stop 컨트롤러 추가.
- **주파수 기반 정렬**: 모든 채널을 실제 라디오 주파수 순으로 재정렬하고 전용 이름 및 주파수 정보 표시.

#### 🛡️ 보안 및 성능
- **사용자 정의 토큰**: HA 애드온 구성 탭에서 직접 보안 토큰을 지정할 수 있는 기능 추가.
- **음질 규격화**: 음질 선택 코드를 0(고음질), 1(보통), 2(절약)로 단일화하여 직관성 개선.

## ⚡ [1.2.0] - 2026-04-22
- **옵션 연동**: 애드온 구성 탭의 `options`와 `schema` 추가 및 `/data/options.json` 동적 로드 구현.

---

## 🛠️ [1.1.0] - 2026-04-21
- **재생 엔진 혁신**: 초고속 WAV/MP3 하이브리드 엔진 도입 (1초 내외 재생 시작).
- **안정성 강화**: 서버(FFmpeg)와 클라이언트(JS) 2중 재연결 및 자동 복구 로직 도입.
- **상태 관리**: 브라우저 종료 후 재접속 시 마지막 채널 및 음질 정보를 유지하는 localStorage 연동.

---

## 🌱 [1.0.0] - 2026-04-20
- **초기 릴리즈**: 기본적인 대한민국 지상파 라디오 스트리밍 기능 구현.

> [!NOTE]
> v2.0.0으로 업데이트 후에는 반드시 애드온 화면에서 **'재빌드(Rebuild)'**를 진행해 주시기 바랍니다.
