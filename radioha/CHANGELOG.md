# Changelog

## 2.2.0
- **보안 강화**: HTTP 보안 헤더(CSP 등) 및 IP 기반 로컬 네트워크 접근 제어 적용.
- **볼륨 제어 추가**: 스피커 재생 시 원격 볼륨 조절 및 음소거 기능 구현.
- **안정성 개선**: 입력값 검증 로직 및 토큰 노출 최소화 적용.

## 2.0.1
- **SBS 버그 픽스**: SBS 방송국 선택시 앱 다운 버그 픽스
  
## 2.0.0
- **멀티 스피커 통합**: 홈어시스턴트 미디어 플레이어 연동 및 스피커 재생 지원.
- **동적 리모컨 UI**: 웹/스피커 출력 기기에 따른 지능형 UI 전환.
- **스마트 제어**: 기기 변경 시 자동 정지 및 전환, 일시정지 기능 추가.
- **채널 정렬**: 주파수 순서 기반 채널 재배치 및 상세 정보(MHz) 표시.
- **보안 강화**: 사용자 정의 보안 토큰 설정 기능 추가.
