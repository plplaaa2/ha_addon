# Changelog

## 2.6.2
- **라이선스 정책 확정**: 방송국 저작권 보호 및 프로젝트 오용 방지를 위해 **개인적/비상업적 용도 전용(Personal/Non-Commercial Use Only)** 라이선스를 최종 적용했습니다.
- **상업적 활용 금지**: 카페, 식당, 기업 등 영리 목적 공간에서의 사용 및 본 소프트웨어를 통한 수익 창출 행위를 엄격히 금지합니다.

## 2.6.1
- **UI/UX 디테일 개선**: 재생 정보 패널의 탐색 버튼 위치를 좌우 끝으로 조정하여 가독성 개선.
- **스크롤 동작 최적화**: 상단 탐색 버튼 사용 시 목록 자동 스크롤 기능을 비활성화하여 시각적 불편함 제거.

## 2.6.0
- **REST API 고도화**: TubePlayer 방식의 POST JSON Body 지원 추가.
- **확장성 강화**: 외부 자동화 도구(Node-RED 등)를 위한 제어 인터페이스 확장.

## 2.5.0
- **채널 탐색 버튼**: 재생 정보 패널에 이전/다음 채널 이동 버튼 추가.
- **자동 스크롤**: 채널 변경 시 목록 내 활성 버튼 위치로 자동 스크롤 기능 구현.

## 2.4.0
- **리모컨 UI 이식**: TubePlayer의 프리미엄 플로팅 바 리모컨 UI(유리 질감, 가로형 볼륨 슬라이더)를 외부 스피커 모드에 이식.

## 2.3.0
- **레이아웃 개선**: 리모컨 모드 사용 시 방송국 목록 하단 겹침 방지를 위한 동적 여백 조절 기능 추가.

## 2.2.0
- **보안 강화**: HTTP 보안 헤더(CSP, X-Frame-Options 등) 적용 및 API 입력값 엄격 검증.
- **IP 기반 보안**: 로컬 네트워크(사설 IP/동일 공인 IP) 접속 시에만 스피커 제어 권한 부여.
- **토큰 노출 최소화**: 전역 변수 주입 방식을 통한 소스 코드 내 토큰 노출 차단.
- **UI 심플화**: 로고 하단 이동 및 텍스트 기반 디자인 개선, 상태바 MHz 정보 제거로 가독성 향상.

## 2.1.1
- **SBS 스트리밍 오류 수정**: SBS 방송 주소 파싱 시 발생하던 `ReferenceError` 및 무한 루프 가능성 해결.
- **안정성 강화**: 채널 매핑 데이터가 없을 경우에 대한 예외 처리 추가.

## 2.1.0
- **프리미엄 UI 개편**: 글래스모피즘(Glassmorphism) 스타일 및 다크 모드 디자인 적용.
- **주파수 디스플레이**: 상단에 대형 주파수 표시 및 실시간 상태(NOW PLAYING/PAUSED/MUTED) 문구 표시.
- **볼륨 제어 강화**: 스피커 재생 시 세로형 볼륨 바 및 스피커 아이콘 기반 음소거(Mute) 기능 추가.
- **모바일 최적화**: 설정 항목 2열 배치 및 요소 크기 조정으로 모바일 사용성 개선.
- **API 확장**: 볼륨 설정(/set_volume) 및 음소거(/mute_volume) 엔드포인트 구현.
- **멀티 스피커 통합**: 홈어시스턴트 미디어 플레이어 연동 및 스피커 재생 지원.
- **동적 리모컨 UI**: 웹/스피커 출력 기기에 따른 지능형 UI 전환.
- **스마트 제어**: 기기 변경 시 자동 정지 및 전환, 일시정지 기능 추가.
- **채널 정렬**: 주파수 순서 기반 채널 재배치 및 상세 정보(MHz) 표시.
- **보안 강화**: 사용자 정의 보안 토큰 설정 기능 추가.
